#  ESCAPE FROM EPSTEIN ISLAND

Escape from Epstein Island adalah permainan petualangan teks (text-based adventure) berbasis Python. Pemain berperan sebagai karakter utama yang terjebak di sebuah pulau misterius bersama 5 teman lainnya. Tantangan utama adalah melarikan diri dalam batas waktu 48 jam dengan cara mengumpulkan tim, mencari item kunci, dan memecahkan misteri pulau tersebut. 

Proyek ini dibuat sebagai tugas akhir (Proyek Akhir Python CLI - Kelas X RPL 1 SMKN 2 JAKARTA) dan kegabutan para kontributornya.
